# 安装包

本目录存放 PinCopy 各平台的预编译安装包，便于直接从 GitHub 下载，无需自行编译。

## Windows（x64）

| 文件 | 说明 |
|------|------|
| [PinCopy_0.1.0_x64-setup.exe](./windows/PinCopy_0.1.0_x64-setup.exe) | NSIS 安装程序（推荐） |

安装前请先退出托盘中的旧版 PinCopy。

## macOS

暂未提供预编译包。macOS 需在 Apple 设备上构建，可参考仓库根目录 [README](../README.md) 中的说明，或通过 GitHub Actions 构建。

## 版本

当前目录中的安装包版本：**0.1.0**

更新安装包时，请同步修改本文件与根目录 README 中的下载链接。
